Definition

  • Cascading (lecture reading occurs from top to bottom) Style Sheet (place with style informations): language that describes how HTML elements are to be displayed on browser/screen, paper, or in other media (its style)
  • Site customization, basically.

    • Concepts

  • Selectors: patterns used to select the element(s) you want to style
  • External Style: the style is written inside an external file
  • Internal Style: the style is written under the style tag (head)
  • Inline Style: the style is written with attributes, on the html tags
  • Pseudo-element: used to style specified parts of an element
  • Pseudo-class: used to define a special state of an element

    • Basic CSS structure summary

    Rule

    A CSS structure consists of a selector and a declaration block. The selector points to the HTML element that will receive style and the declaration block contains one or more declarations separated by semicolons

    A declaration block includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces

    class and id rule: attributes that helps us, in case we want to apply a style to a unique tag or more than one

  • id="name": only one tag can have this id, and to apply css, the style tag must contain the hash (#) before the selector
  • class="name": apply the same style to multiple elements, and to apply the css, the style tag must contain a dot (.) before the selector

    • [selector/element target] + [with or without .class/#id] + { property: property value; ... }

    External file


    1. Create css file and name it

    2. Inside the html file, put under the head tag a link tag, with rel, type and href attributes

    3. Fill the rel attribute with rel="stylesheet"

    4. Fill the type attribute with type="text/css"

    5. Fill the href attribute with href="css_file_name.css"
